HTML Tables Cheat Sheet

Elements

Tag Name Description
<table> Table
  • Creates a table.
  • Has content that is used to represent a 2D table made of rows and columns.
<thead> Table Head
  • Defines the headings of table columns encapsulated in table rows.
  • The set of rows defining the column headers in a table.
<tr> Table Row
  • Used to add rows to a table before adding table data and table headings.
<th> Table Heading
  • Used to add titles to rows and columns of a table (clarify the meaning of data).
  • Must be enclosed in table row elements.
<td> Table Data
  • Can be nested inside a table row element to add a cell of data to a table.
<tbody> Table Body
  • Tables can be split into 3 main sections: a head (see above), a body, and a footer (see below).
  • Semantic HTML element that will contain all table data other than table heading and footer content (i.e. the rows containing the actual table data).
  • If used, will contain all table row elements (these make up the body of the table).
  • <table> can't have both <tbody> and <tr> as direct children.
<tfoot> Table Footer
  • Uses table rows to give footer content, or to summarise content at the end of a table.

Attributes

Tag Name Description
rowspan Row Span
  • Indicates how many rows a cell/<td> element should span within the table.
  • Accepts an integer (greater than or equal to 1) to denote the number of rows it spans across.
  • Default value of 1, and will take any positive integer up to 65534).
colspan Column Span
  • Indicates how many columns a cell/<td> or <th> element should span within the table.
